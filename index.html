git-svn更换服务器域名

  1. 更换地址

    更换完域名后测试已不能更新

  2. 切换到git-svn(git 默认给svn创建的版本)

  3. 此时需要在svn主库上做一个提交,可以让其他用户随便修改或添加一个提交

  4. 等其他用户提交完后我们在git-svn分支做一个更新

  5. 切换到master

  6. 执行 git rebase git-svn

  7. 查看日志已经是最新的了

  8. 执行 rebase已经不再报错了
